SIMPLE_ENCRYPTION

Simple encryption and hashing for Eiffel

Overview Quick Start API Reference Security GitHub

Overview

SIMPLE_ENCRYPTION provides secure hashing and password management for Eiffel applications, wrapping the ISE EEL library.

Key Features

  • SHA-256 hashing
  • HMAC-SHA256 message authentication
  • PBKDF2 password hashing
  • Secure password verification
  • Random token generation

Quick Start

<library name="simple_encryption" location="$SIMPLE_ENCRYPTION/simple_encryption.ecf"/>

Password Hashing

local
    enc: SIMPLE_ENCRYPTION
    stored_hash: STRING
do
    create enc.make

    -- Hash password for storage
    stored_hash := enc.hash_password ("user_password")
    -- Returns: salt$iterations$hash

    -- Verify password
    if enc.verify_password ("user_password", stored_hash) then
        print ("Access granted%N")
    end
end

SHA-256 Hashing

hash := enc.sha256 ("data to hash")
-- Returns 64-character hex string

Random Generation

FeatureDescription
random_bytes (count)Random bytes

random_token (length)URL-safe token

Encoding

FeatureDescription
bytes_to_hex (bytes)Bytes to hex
hex_to_bytes (hex)Hex to bytes

Security Notes

  • PBKDF2 uses 600,000 iterations by default (OWASP 2025)
  • Password verification uses constant-time comparison
  • Random generation is time-seeded (not cryptographically secure)
  • For production, consider hardware random sources
